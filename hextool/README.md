A set of convenient little tools for eurorack module, synth and pedal-makers. 
**M7** works for Thonk (or other brands) subminiature switches,
**M8** works for jack nuts (only hex, not the knurled variant) and regular switches,
**M10** works for potentiometers

The 3MF file includes a print profile for the Bambu Labs A1 mini. 

Make your own / customize using [Onshape](https://cad.onshape.com/documents/79d5f8374dd82f0186759913/w/4e72a149bc711f31f4bd8f6c/e/3a4084fc5f19f3a475d38b66?renderMode=0&rightPanel=variableTablePanel&uiState=69e5dfb72a32b867e8c59061) 
Use the variable table on the right and edit per the given instructions.
